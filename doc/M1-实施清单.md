# hi-im-gateway M1 实施清单

> **对应生态里程碑**：档 C **M5**（依赖 **M4** hi-im-usrsvr + hi-im-seqsvr）  
> **工期**：约 2 周（全职）  
> **前置**：hi-im-api / hi-im-hubclient **v0.1.0**；hi-im-usrsvr **v0.1.0**（register + iplist + ONLINE）；hi-im M3 Hub 可冒烟

---

## 目标

交付 **WebSocket 接入 + 注册上线全链路**：FORWARD hubclient、WS 二进制 IM 帧、ONLINE 上下行、ChatTab 会话表、双 gateway 实例。**不含** msgsvr 群聊第一段、**不含** ImGroup 群 fan-out（M6）。

**M5 总验收**（档 C §11.3）：

| 项 | 标准 |
|----|------|
| 链路 | register → iplist → WS → ONLINE → ONLINE_ACK → LOGIN |
| 双窗口 | 两个 gateway NID 均可上线（可选 echo/PING） |
| 边界 | **不**验收 GROUP-CHAT 群聊互通 |

## 常见误区（档 C 对齐）

| 误区 | 正确 |
|------|------|
| gateway 连 BACKEND | 接入层连 **FORWARD** |
| gateway 做 register/AllocSid | **usrsvr** HTTP + seqsvr |
| M5 验收双窗口群聊 | 群聊 **M6**（需 msgsvr + ImGroup） |
| gateway SUB GROUP-CHAT 做 fan-out | **msgsvr** SUB；gateway 只做 **第二段** ImGroup |
| 在 gateway 写 Redis 会话 | **usrsvr** ONLINE 写 Redis |
| M3 主仓冒烟测 WS | M3 仅 hub+stub；WS **M5** |
